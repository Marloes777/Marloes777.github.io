Welkom bij de Examen Monitor!


Hier kan je zien hoe ik voor mijn examen sta.
Onder het kopje "Over mij" kan je lezen wie ik ben, onder het kopje "Cijfers" kan je zien hoe ik voor mijn examen sta en het gemiddelde van cijfers berekenen.
Onder "Vervolgopleiding" kan je vinden wat voor vervolgopleiding ik wil gaan doen.



De Examen Monitor is een webiste waar ik overzichtelijk op kan zien hoe ik voor mijn examen sta, wat mijn cijfers zijn, voor welke vakken ik harder moet werken dan de anderen, en waar ik mijn cijfers kan berekenen. Ook kan ik bijhouden welke vervolgopleidingen mij leuk lijken. Zo is het makkelijker om een weergave te krijgen van hoe ik voor mijn examen sta.
